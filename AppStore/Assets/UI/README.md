# UI Design Assets

This folder should contain:
- App Store preview images (optional)
- Marketing graphics (optional)

## Required Design Files for App Store
1. **App Icon** (1024x1024 PNG) → stored in `../Icon/`
2. **Screenshots** → stored in `../../AppStoreScreenshots/`

## Optional
- Figma/Sketch source files for App Store assets
- Marketing banner images
- Feature graphic (for Chinese App Store)
